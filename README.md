<h1>Burger App</h1>

## Installation

```
git clone https://github.com/brainstormchaser/burger.git
cd burger
npm install
```

## Available Node Commands and URLs

**Local:** 

* Run `node server.js` 
* Then load [http://localhost:6060/](http://localhost:6060/)

## Misc Notes

* Github repo pushes autodeploy to Heroku.
